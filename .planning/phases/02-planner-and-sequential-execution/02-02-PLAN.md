1. Create `src/git/worktree-manager.ts`:
   - Import `simpleGit, type SimpleGit` from `simple-git`
   - Import `join` from `node:path`
   - Import `rm` from `node:fs/promises`

   Export class `WorktreeManager`:
   ```
   private git: SimpleGit
   private baseDir: string
   private runId: string
   private activeWorktrees: Map<string, { worktreePath: string; branch: string }>
   ```
   - Constructor takes `baseDir: string` and optional `runId?: string` (defaults to `crypto.randomUUID()`)
   - `this.git = simpleGit(baseDir)`
   - `this.activeWorktrees = new Map()`

   Methods:
   - `async create(taskId: string): Promise<{ worktreePath: string; branch: string }>`
     - `branch = \`anvil/run-${this.runId}/task-${taskId}\``
     - `worktreePath = join(this.baseDir, '.anvil', 'worktrees', \`task-${taskId}\`)`
     - `await this.git.raw(['worktree', 'add', worktreePath, '-b', branch])`
     - Store in `this.activeWorktrees.set(taskId, { worktreePath, branch })`
     - Return `{ worktreePath, branch }`

   - `async commitAndMerge(taskId: string, message: string): Promise<void>`
     - Get `{ worktreePath, branch }` from `this.activeWorktrees` (throw if not found)
     - `const worktreeGit = simpleGit(worktreePath)`
     - `await worktreeGit.add('.')`
     - Check if there are changes: `const status = await worktreeGit.status()` — if no changes, skip commit
     - `await worktreeGit.commit(message)`
     - Merge back: `await this.git.merge([branch, '--no-ff'])`

   - `async cleanup(taskId: string): Promise<void>`
     - Get info from activeWorktrees (return silently if not found — defensive for signal handlers)
     - `const worktreePath = info.worktreePath`
     - Try: `await this.git.raw(['worktree', 'remove', worktreePath, '--force'])`
     - Catch: fallback to `await rm(worktreePath, { recursive: true, force: true })` then `await this.git.raw(['worktree', 'prune'])`
     - Try: `await this.git.branch(['-D', info.branch])`
     - Catch: swallow (branch may already be deleted)
     - `this.activeWorktrees.delete(taskId)`

   - `async pruneStale(): Promise<void>`
     - `await this.git.raw(['worktree', 'prune'])`

   - `async cleanupAll(): Promise<void>`
     - For each taskId in activeWorktrees, call `this.cleanup(taskId)`

   Export standalone function `validateTouchMap(worktreePath: string, writes: string[]): Promise<{ valid: boolean; violations: string[] }>`:
   - `const worktreeGit = simpleGit(worktreePath)`
   - Get staged + unstaged changes: `const diff = await worktreeGit.diff(['--name-only', 'HEAD'])`
   - Get untracked: `const untracked = await worktreeGit.raw(['ls-files', '--others', '--exclude-standard'])`
   - Combine into `allChanged` array, filter empty strings
   - `const writesSet = new Set(writes)`
   - `const violations = allChanged.filter(f => !writesSet.has(f))`
   - Return `{ valid: violations.length === 0, violations }`

   - Import `Anthropic` from `@anthropic-ai/sdk`
   - Import `type Task` from `../schemas/plan.js`
   - Import `type AnvilConfig` from `../schemas/config.js`
   - Import `validateTouchMap` from `../git/worktree-manager.js`
   - Import `WORKER_SYSTEM_PROMPT, WORKER_TOOLS` from `../prompts/worker-system.js`

   Export interface `WorkerResult`:
   ```typescript
   export interface WorkerResult {
     taskId: string;
     success: boolean;
     filesWritten: string[];
     error?: string;
   }
   ```

   Export `async function executeTask(task: Task, worktreePath: string, config: AnvilConfig, options?: { client?: Anthropic }): Promise<WorkerResult>`:
   - Create client: `options?.client ?? new Anthropic()`
   - Build user message:
     - Include task description
     - Include writes[] and reads[] lists
     - Include acceptance criteria
     - For each file in `task.reads`, try to read from `join(worktreePath, file)` and include content as context (wrapped in fenced code blocks with file path heading). Catch and skip if file doesn't exist.
   - Call `client.messages.create()` with:
     - `model: config.model`
     - `max_tokens: 16384`
     - `system: WORKER_SYSTEM_PROMPT`
     - `messages: [{ role: 'user', content: userMessage }]`
     - `tools: WORKER_TOOLS as any` (the const assertion needs casting for SDK compatibility)
   - Process response: iterate over `response.content` blocks
     - For `tool_use` blocks with `name === 'write_file'`: extract `input.path` and `input.content`, write to `join(worktreePath, input.path)` (create parent dirs with `mkdir(dirname(...), { recursive: true })`)
     - For `tool_use` blocks with `name === 'report_error'`: return `{ taskId: task.id, success: false, filesWritten: [], error: input.reason }`
     - Track written files in a `filesWritten: string[]` array
   - After processing all tool calls, validate touch map:
     - `const touchResult = await validateTouchMap(worktreePath, task.writes)`
     - If `!touchResult.valid`, return `{ taskId: task.id, success: false, filesWritten, error: \`Touch map violation: files modified outside writes[]: ${touchResult.violations.join(', ')}\` }`
   - Return `{ taskId: task.id, success: true, filesWritten }`
