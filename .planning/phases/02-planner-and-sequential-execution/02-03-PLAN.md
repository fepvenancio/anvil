   Export `async function executeSequentially(plan: Plan, config: AnvilConfig, options?: { client?: Anthropic; baseDir?: string }): Promise<ExecutionResult>`:
   - `const baseDir = options?.baseDir ?? process.cwd()`
   - `const worktreeManager = new WorktreeManager(baseDir)`
   - `await worktreeManager.pruneStale()` (clean any stale worktrees from previous runs)
   - `const ordered = topologicalSort(plan.tasks)`
   - `const results: WorkerResult[] = []`
   - `const failedTasks: string[] = []`

   - Register signal handlers for cleanup:
     ```
     const cleanup = async () => {
       console.log('\nCleaning up worktrees...');
       await worktreeManager.cleanupAll();
       process.exit(1);
     };
     process.on('SIGINT', cleanup);
     process.on('SIGTERM', cleanup);
     ```

   - For each task in ordered:
     - Print: `chalk.blue(\`[${i+1}/${ordered.length}] Executing task: ${task.id}\`)`
     - Print: `chalk.dim(\`  ${task.description.slice(0, 100)}\`)`
     - `const { worktreePath } = await worktreeManager.create(task.id)`
     - Try:
       - `const result = await executeTask(task, worktreePath, config, { client: options?.client })`
       - `results.push(result)`
       - If `result.success`:
         - `await worktreeManager.commitAndMerge(task.id, \`feat(anvil): ${task.description.slice(0, 72)}\`)`
         - Print: `chalk.green(\`  ✓ Task ${task.id} complete (${result.filesWritten.length} files)\`)`
       - Else:
         - `failedTasks.push(task.id)`
         - Print: `chalk.red(\`  ✗ Task ${task.id} failed: ${result.error}\`)`
         - Break the loop (sequential mode: stop on first failure)
     - Finally:
       - `await worktreeManager.cleanup(task.id)`

   - Remove signal handlers
   - Save plan to `.anvil/roadmap.json`: `await writeFile(join(baseDir, '.anvil', 'roadmap.json'), JSON.stringify(plan, null, 2))`
   - Return `{ success: failedTasks.length === 0, results, failedTasks }`
